1. download agent file.
2. move .jar file to home folder. // not required
3. add hosts to hosts file.
https://www.howtogeek.com/howto/27350/beginner-geek-how-to-edit-your-hosts-file/

4. open php storm open edit VM option.
5. set -javaagent:<path> (path to home .jar file).
6. restart PhpStorm.
7. open help -> register and add activate key from ket.txt file.

PhpStorm version 2021.2.1
